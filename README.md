# Checkout the REV2 PCB:

[link to rev2](/rev2/README.md)

### <u>Features:</u>

- Atmega32U2 as MCU

- north-facing Kailh-Hotswap support 

- south-facing solder options

- USB-C

- 

- 3d printed case with gasket mount 

- options for WKL,HHKB, WKL+HHKB

- split spacebar

- support for QMK+Vial/Via
